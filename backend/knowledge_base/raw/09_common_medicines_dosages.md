# Common Poultry Medicines, Dosages & Withdrawal Periods — Bangladesh

## Important Safety Notice

⚠️ **Always consult a veterinarian before administering medicines.**
⚠️ **Maintain withdrawal periods strictly before selling meat or eggs.**
⚠️ **Never use antibiotics as growth promoters — this is illegal and harmful.**
⚠️ **Only 14.3% of farmers in Bangladesh know about withdrawal periods — be responsible.**

## Antibiotics

### 1. Amoxicillin

| Parameter | Detail |
|-----------|--------|
| Type | Broad-spectrum penicillin |
| Indications | E. coli, Salmonellosis, CRD secondary infections, respiratory infections |
| Dosage (water) | 20mg per kg body weight, or 1g per 4 liters of drinking water |
| Dosage (feed) | 250-500g per ton of feed |
| Duration | 3-5 days |
| Route | Drinking water or feed |
| Withdrawal — Meat | 7 days |
| Withdrawal — Eggs | 0 days |
| Available brands (BD) | Amoxivet, Moxacil Vet, Amoxicure |

## Anticoccidials

### 1. Amprolium

| Parameter | Detail |
|-----------|--------|
| Type | Thiamine analogue (non-antibiotic) |
| Indications | Coccidiosis — treatment and prevention |
| Treatment dose | 250mg per liter of drinking water for 5-7 days |
| Prevention dose | 125mg per liter or 0.5g per kg feed |
| Route | Drinking water or feed |
| Withdrawal — Meat | 1 day |
| Withdrawal — Eggs | 0 days |
| Available brands (BD) | Amprol Vet, Coxiprol, Amprolin |
| Note | **Safest anticoccidial — preferred first choice** |

### 2. Toltrazuril

| Parameter | Detail |
|-----------|--------|
| Type | Triazinetrione |
| Indications | Coccidiosis — treatment (all Eimeria species) |
| Dosage | 7mg per kg body weight (single day treatment) |
| Practical dose | 25ml of 2.5% solution per liter of drinking water for 2 consecutive days |
| Route | Drinking water |
| Withdrawal — Meat | 14 days |
| Withdrawal — Eggs | Not for use in laying hens |
| Available brands (BD) | Baycox, Coxinil, Tolcox Vet |
| Note | Very effective single-dose treatment |

### 3. Sulfonamides (Sulfadimethoxine)

| Parameter | Detail |
|-----------|--------|
| Indications | Coccidiosis, Fowl Cholera |
| Dosage | 50mg per kg body weight |
| Duration | First 2 days full dose, then half dose for 3 more days |
| Withdrawal — Meat | 5 days |
| Withdrawal — Eggs | 10 days |

### Feed Anticoccidials (Preventive — for broilers)

| Drug | Dose in Feed | Notes |
|------|-------------|-------|
| Salinomycin | 60-70g per ton | Ionophore, commonly used |
| Monensin | 100-120g per ton | Ionophore |
| Lasalocid | 75-125g per ton | Ionophore |
| Nicarbazin | 100-125g per ton | Chemical, withdraw 5 days before slaughter |
| Diclazuril | 1g per ton | Chemical |

**Shuttle Program**: Rotate between ionophore and chemical anticoccidials each batch to prevent resistance development.

## Vitamins and Supplements

### Vitamin C (Ascorbic Acid)
| Parameter | Detail |
|-----------|--------|
| Primary use | Heat stress management |
| Dose in water | 200-300mg per liter |
| Dose in feed | 200-500mg per kg |
| When to use | During heat stress (>32°C), disease recovery, post-vaccination |
| No withdrawal period |

### Vitamin AD₃E
| Parameter | Detail |
|-----------|--------|
| Primary use | General health, immune support, reproduction |
| Dose | As per product label (typically 1ml per liter for 3-5 days) |
| When to use | During stress, disease recovery, post-vaccination, before/during laying |
| Vitamin A | Helps tissue repair (especially respiratory and gut mucosa) |
| Vitamin D₃ | Calcium absorption, eggshell quality |
| Vitamin E | Antioxidant, immune function |

### Vitamin K
| Parameter | Detail |
|-----------|--------|
| Primary use | Blood clotting (for coccidiosis, hemorrhagic conditions) |
| Dose | 2-3mg per kg feed, or as per product label in water |
| When to use | During coccidiosis (bloody droppings), hemorrhagic diseases |

### Electrolyte Solutions
| Parameter | Detail |
|-----------|--------|
| Primary use | Dehydration, heat stress, diarrhea |

| Homemade formula | 1 teaspoon salt + 1 tablespoon sugar + ½ teaspoon baking soda per liter of water |
| When to use | Heat stress, diarrhea, post-vaccination stress, transport stress |

### Liver Tonic / Hepatoprotectant
| Parameter | Detail |
|-----------|--------|
| Primary use | Liver support during antibiotic therapy, mycotoxin exposure |
| Active ingredients | Choline chloride, methionine, biotin, sorbitol |
| Dose | As per product label |
| When to use | After prolonged antibiotic use, suspected aflatoxin exposure |

### Probiotics
| Parameter | Detail |
|-----------|--------|
| Primary use | Gut health, competitive exclusion of pathogens |
| Active organisms | Lactobacillus, Bacillus subtilis, Saccharomyces |

## Anthelmintics (Deworming)

### Levamisole
| Parameter | Detail |
|-----------|--------|
| Indications | Roundworms (Ascaridia galli), Heterakis |
| Dosage | 18-36mg per kg body weight (single dose) |
| Route | Drinking water (dissolve in water for 4-6 hours) |
| Withdrawal — Meat | 7 days |
| Withdrawal — Eggs | 3 days |
| Frequency | Every 2-3 months |

### Piperazine
| Parameter | Detail |
|-----------|--------|
| Indications | Roundworms |
| Dosage | 200-300mg per kg body weight |
| Route | Drinking water or feed |
| Withdrawal — Meat | 14 days |
| Withdrawal — Eggs | 0 days |

### Albendazole
| Parameter | Detail |
|-----------|--------|
| Indications | Roundworms, tapeworms |
| Dosage | 10-20mg per kg body weight |
| Route | Oral (individual dosing preferred) |
| Withdrawal — Meat | 14 days |
| Withdrawal — Eggs | 7 days |

### Deworming Schedule
- Deworm every **2-3 months** for free-range and semi-scavenging birds
- For confined birds: every **3-4 months**
- Rotate between drug classes to prevent resistance
- Deworm at least 2 weeks before vaccination for better immune response

## Disinfectants

| Disinfectant | Concentration | Use | Contact Time |
|-------------|---------------|-----|-------------|
| Formalin (formaldehyde 37%) | 2-3% solution | Shed surfaces, equipment | 30 minutes |
| Quaternary ammonium (QAC) | 0.1-0.2% | Shed surfaces, footbaths | 10-15 minutes |
| Sodium hypochlorite (bleach) | 0.5% (5000 ppm) | Surfaces, water lines | 10 minutes |
| Potassium permanganate | 1:1000 solution | Footbaths, wound cleaning | Continuous |
| Glutaraldehyde | 2% | Equipment, cold disinfection | 10-30 minutes |
| Lime (quicklime/CaO) | Whitewash | Walls, floors, dead bird pit | Apply thick coat |
| Povidone-iodine | 1-2% | Wound cleaning, navel dressing | Apply directly |

### Footbath Protocol
- Place at every shed entrance
- Fill with 2-3% formalin or QAC solution
- Change solution daily (or when visibly dirty)
- Ensure boots are submerged for at least 30 seconds
- Scrub visible dirt off boots before dipping

## Banned / Restricted Substances

The following should **NEVER** be used in poultry in Bangladesh:

| Substance | Reason |
|-----------|--------|
| Chloramphenicol | Banned — aplastic anemia risk in humans |
| Nitrofurans (Furazolidone, Furaltadone) | Banned — carcinogenic residues |
| Diethylstilbestrol (DES) | Banned growth hormone |
| Clenbuterol | Banned — not approved for poultry |
| Ractopamine | Banned — not approved |
| Melamine | Illegal feed adulterant |
| Arsenic compounds | Banned growth promoters |

**Using banned substances is a criminal offense and puts consumers at serious health risk.**
